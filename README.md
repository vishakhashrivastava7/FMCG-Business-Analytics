# FMCG Business Insights & Inventory Analytics using Python

## Project Overview

This project analyzes FMCG sales, product, and inventory data to generate business insights, identify inventory risks, evaluate product performance, and support data-driven decision making.

## Objectives

* Analyze sales and profit performance.
* Identify top-performing products and categories.
* Monitor inventory levels and stock risks.
* Evaluate regional sales performance.
* Generate AI-assisted demand forecasting insights.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook
* Power BI

## Key Analysis Performed

### Sales Analysis

* Total Revenue Analysis
* Monthly Sales Trend Analysis
* Regional Performance Analysis
* Product Performance Analysis

### Profitability Analysis

* Category-wise Profit Analysis
* Top Products by Profit

### Inventory Analysis

* Inventory Value Calculation
* Low Stock Product Identification
* Stock Status Monitoring
* Inventory Risk Assessment

### AI-Powered Forecasting

* Linear Regression based sales forecasting
* Future demand estimation
* Business planning support

## Key Insights

* Identified top revenue-generating products and categories.
* Detected low-stock products requiring replenishment.
* Evaluated regional sales contribution and profitability.
* Estimated future sales demand using predictive analytics.

## Business Impact

The project demonstrates how analytics can improve inventory planning, product strategy, and operational decision-making in FMCG businesses.
